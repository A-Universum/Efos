# Efos
Efos — интеллектуальная система, основанная на онтологическом подходе. Efos моделирует предметную область в виде семантического графа, где объекты представлены узлами, а связи между ними — рёбрами. Такая архитектура позволяет не просто хранить данные, а выявлять скрытые закономерности и принимать решения с учётом контекста и смысловых отношений
